## D. 冷熱欄位判定（每兩週更新）

### 熱欄位（Hot）

符合任一條件即列入：

- 查詢條件使用率 >= 5%
- 排序使用率 >= 3%
- 業務核心流程必用（例如：provider/country/exportStatus/updatedAt）
- 慢查詢中反覆出現

### 溫欄位（Warm）

- 使用率 1% ~ 5%
- 偶爾作為過濾條件

### 冷欄位（Cold）

- 使用率 < 1%
- 主要只展示，幾乎不過濾/排序
